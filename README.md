# Pr-ctica-Tema-3---Instal-laci-i-Configuraci-de-WordPress

## Descripcion del projecto
Para llevar a cabo este proyecto, creé un portal de noticias sobre fútbol llamado 
**“LA GAMBETA”**, utilizando **"WordPress"**. El objetivo era desarrollar una página web funcional donde se pudieran publicar y organizar noticias de lo que quisieras yo escogi de futbol

Primero, instalé y configuré WordPress  (Pulse News). Después, personalicé el diseño del portal modificando elementos como el menú de navegación, los colores, las secciones y la página principal, con el fin de que fuera clara y fácil de usar.

A continuación, creé diferentes categorías de noticias (por ejemplo: partidos, fichajes, ligas o equipos) para organizar correctamente el contenido. También publiqué varios artículos de prueba, añadiendo títulos, textos, imágenes destacadas y etiquetas, simulando el funcionamiento de un portal real de noticias.

Durante el proceso también configuré elementos clave del portal, como:

El menú de navegación.

La página de inicio.

Las publicaciones del blog.

La gestión de contenidos desde el panel de administración.

La página está organizada en distintas secciones:

Últimas 24 h: para mostrar las noticias más recientes de la Gambeta.

Análisis de los mejores equipos: donde se publican artículos sobre todo lo que tenga que ver con el futbol.

Champions League: secciones dedicadas a las competiciones, con noticias, resultados.

Por último, comprobé que la página funcionara correctamente tanto en ordenadores como en dispositivos móviles, asegurando que el diseño fuera adaptable y fácil de navegar para los usuarios.
